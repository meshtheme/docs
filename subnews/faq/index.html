Installation & Setup

Q: Do I need any special hosting to use SubNews?

A: SubNews works on standard WordPress hosting. You need:

  • WordPress 5.8 or higher
  • PHP 7.4 or higher
  • At least 256 MB memory limit (512 MB recommended)
  • MySQL 5.6 or higher

Most modern hosting providers offer these specifications by default.

Q: What plugins are required?

A: Four plugins are required:

  1. WPBakery Page Builder - Page building
  2. Mesh Core - Custom shortcodes and elements
  3. Redux Framework - Theme options management
  4. CMB2 - Custom metaboxes and fields

These should be installed automatically after activating SubNews. If not, install manually from Plugins menu.

Q: Can I use SubNews with a child theme?

A: Yes! Child themes are recommended for customization. A child theme:

  • Keeps your customizations safe during updates
  • Allows easy theme updates
  • Best practice for WordPress

See Getting Started guide for child theme setup.

Q: Where do I find the theme options?

A: Go to Appearance → SubNews Theme Options in WordPress dashboard. All theme customization options are there.

Q: Can I import demo content?

A: Yes! Install “One Click Demo Import” plugin, then go to Appearance → Import Demo Data. Choose a demo and import. Takes 2-5 minutes depending on content size.

Q: Where do I enter my license key?

A: After purchasing SubNews:

  1. Go to Appearance → SubNews Theme Options
  2. Click Theme License tab
  3. Enter your purchase code from ThemeForest
  4. Click Activate License
  5. You’ll receive confirmation message

License activation enables automatic updates and premium support.

Q: How do I update the theme?

A: Two methods:

Automatic Updates (Recommended):

  1. Activate your license key (see above)
  2. Go to Dashboard → Updates
  3. Check for SubNews updates
  4. Click Update button

Manual Updates:

  1. Download latest version from ThemeForest
  2. Go to Appearance → Themes
  3. Upload new theme file
  4. Existing settings preserved automatically

Always backup before updating!

Q: Why am I not getting update notifications?

A: Common reasons:

  1. License key not activated
  2. License expired or used on multiple sites
  3. WordPress not checking for updates
  4. Check Dashboard → Updates manually
  5. Verify license status in Theme Options

If issue persists, contact support with purchase code.

Plus 9 different “block” styles. Change via SubNews Theme Options → Blog.

A: Yes! SubNews is fully compatible with WooCommerce. Install WooCommerce plugin and configure product settings in SubNews Theme Options → WooCommerce.

Configure in SubNews Theme Options → Portfolio.

  • News blocks

Lite Slider

Q: How do I use the Lite Slider?

A: The Lite Slider creates beautiful full-width sliders:

  1. Go to SubNews → Lite Slider
  2. Click Add New Slider to add slides
  3. For each slide:
    • Upload background image or video
    • Add title and description text
    • Set button links (optional)
    • Choose text position and colors
  4. Configure slider controls (height, animation, timing)
  5. Copy shortcode: [mtc_lite_slider]
  6. Paste shortcode on any page or post

See Lite Slider Guide for detailed instructions.

Q: My slider images are cut off on mobile

A: Use the Mobile/Tablet Image Focal Point setting:

  1. Edit the slide in Lite Slider settings
  2. Find Set the Image Focal Point for Mobile/Tablet
  3. Drag the red dot to the important area of your image
  4. Or enter custom position manually (e.g., “50% 30%” for center-top)
  5. Save changes

This ensures important content stays visible on smaller screens.

Q: Can I add videos to Lite Slider?

A: Yes! The Lite Slider supports video backgrounds:

  1. Upload video files: MP4, WebM, WebP, OGG, WMV, MOV, or AVI
  2. For best compatibility, use MP4 format
  3. Keep file size under 10MB for performance
  4. Videos auto-play on desktop (muted for autoplay compatibility)
  5. Falls back to poster image on mobile devices

Q: How do I change slider height?

A: In Lite Slider control options:

  1. Go to SubNews → Lite Slider
  2. Scroll to Slider Control Options
  3. Find Slider Height field
  4. Enter value with unit (e.g., 550px, 80vh, or 100vh for fullscreen)
  5. Default is 550px
  6. Use 100vh for full-screen hero sliders

Q: Slider arrows or indicators not showing

A: Check your slider settings:

  1. Go to SubNews → Lite Slider
  2. Check Enable Slider Arrow is set to “Show”
  3. Check Enable Slider Indicators is set to “Show”
  4. Clear browser cache and refresh page
  5. Check if custom CSS is hiding controls
  6. Verify JavaScript is not blocked

Q: How do I disable slider autoplay?

A: To stop automatic sliding:

  1. Go to SubNews → Lite Slider
  2. Find Slide Interval Time field
  3. Enter 0 (zero) to disable autoplay
  4. Slider will only change when user clicks arrows/indicators
  5. Good for sliders with important content to read

Mega Menu & Navigation

Q: How do I set up a mega menu?

A: SubNews supports mega menus with custom content:

  1. Go to Appearance → Menus
  2. Create or select your menu
  3. Add menu items as usual
  4. For mega menu columns, add custom content:
    • Use Custom Links for column headers
    • Use Static Blocks for rich content
    • Drag items to create sub-levels
  5. Assign menu to Primary Menu location
  6. Enable mega menu in SubNews Theme Options → Header → Mega Menu

Q: Can I add images to mega menu?

A: Yes! Two methods:

Method 1: Static Blocks

  1. Create Static Block with images/content
  2. Add block shortcode to menu item description
  3. Rich content displays in mega menu

Method 2: Category Images

  1. Add category with featured image
  2. Add category to menu
  3. Category image displays automatically

See Navigation Guide for examples.

Q: Menu items not clickable on mobile

A: Troubleshooting steps:

  1. Check if JavaScript is loading properly
  2. Disable browser extensions temporarily
  3. Clear browser cache completely
  4. Check for JavaScript errors in browser console
  5. Test in different mobile browser
  6. Verify mobile menu is enabled in theme options
  7. Check for CSS z-index conflicts

Widgets & Sidebars

Q: How do I add widgets to sidebar?

A: Adding widgets is simple:

  1. Go to Appearance → Widgets
  2. Find available widgets on left side
  3. Choose sidebar/footer area on right side
  4. Drag widget to desired area
  5. Configure widget settings
  6. Click Save

See Sidebar Widgets Guide for widget options.

Q: Can I create custom sidebars?

A: Yes! SubNews supports unlimited custom sidebars:

  1. Go to Appearance → SubNews Theme Options → Sidebar
  2. Click Add New Sidebar
  3. Enter sidebar name
  4. Click Add
  5. New sidebar appears in Appearance → Widgets
  6. Assign sidebar to pages via page metaboxes

Useful for different sections with unique widgets.

Q: Widgets not showing on my page

A: Check these settings:

  1. Verify page layout has sidebar enabled (not “No Sidebar”)
  2. Check widgets are added to correct sidebar area
  3. Confirm sidebar is assigned to page in metaboxes
  4. Clear all caches (plugin, browser, server)
  5. Check widget visibility settings
  6. Verify conditional widget logic if using plugins

Q: How do I remove widgets from specific pages?

A: Two options:

Option 1: Change Page Layout

  1. Edit the page
  2. Find Page Options metabox
  3. Select No Sidebar layout
  4. Update page

Option 2: Conditional Widgets

  1. Install widget visibility plugin
  2. Configure conditions per widget
  3. Hide widgets on specific pages/categories

Mobile & Responsive

Q: Is SubNews mobile-friendly?

A: Yes! SubNews is fully responsive:

  • Adapts to all screen sizes automatically
  • Touch-optimized navigation
  • Optimized images for mobile
  • Fast loading on mobile networks
  • Mobile-first design approach
  • Tested on iOS and Android devices

Google Mobile-Friendly Test approved.

Q: How do I test mobile responsiveness?

A: Multiple testing methods:

Browser Testing:

  1. Open your site in Chrome/Firefox
  2. Press F12 to open Developer Tools
  3. Click device toolbar icon
  4. Test different device sizes

Real Device Testing:

  1. Open site on actual phone/tablet
  2. Test navigation and interactions
  3. Check loading speed
  4. Verify images display correctly

Online Tools:

  • Google Mobile-Friendly Test
  • Responsive Design Checker
  • BrowserStack for multiple devices

Q: Mobile menu not opening

A: Troubleshooting steps:

  1. Clear browser cache on mobile device
  2. Check JavaScript is enabled
  3. Verify jQuery is loading (check browser console)
  4. Disable caching plugins temporarily
  5. Test in different mobile browser
  6. Check for JavaScript errors in console (F12)
  7. Ensure no conflicting plugins

Q: Images too large on mobile

A: Optimize for mobile viewing:

  1. Use Focal Point setting for slider images
  2. Upload appropriately sized images (not larger than needed)
  3. Use responsive image plugins
  4. Enable lazy loading
  5. Check SubNews Theme Options → Performance
  6. Consider using WebP format for better compression

Q: Text too small on mobile

A: Adjust typography settings:

  1. Go to SubNews Theme Options → Typography
  2. Set appropriate font sizes for mobile
  3. Use relative units (em, rem) for better scaling
  4. Test on actual devices
  5. Adjust line height for readability
  6. Check WordPress Settings → Reading for site width

  • Clean semantic HTML5 markup

  • Schema.org microdata support
  • Optimized heading structure
  • SEO-friendly URLs
  • Social media meta tags

Translation & Multilingual

Q: Can I translate SubNews to other languages?

A: Yes! SubNews is translation-ready:

Method 1: Translation Plugins

  1. Install WPML or Polylang
  2. Configure languages
  3. Translate content page by page
  4. Automatic language switcher

Method 2: .po/.mo Files

  1. Use Poedit software
  2. Open /languages/subnews.pot file
  3. Translate strings
  4. Save as subnews-{locale}.mo
  5. Upload to /wp-content/languages/themes/

Method 3: Loco Translate Plugin

  1. Install Loco Translate
  2. Select SubNews theme
  3. Add new language
  4. Translate in WordPress dashboard

Q: Is SubNews compatible with WPML/Polylang?

A: Yes! SubNews is fully compatible with:

  • WPML - Complete compatibility, tested thoroughly
  • Polylang - Full support for all features
  • TranslatePress - Works seamlessly
  • qTranslate-X - Compatible

All theme options, widgets, and custom content can be translated.

Q: Does SubNews support RTL languages?

A: Yes! SubNews includes RTL (Right-to-Left) support:

  • Automatic RTL stylesheet loading
  • RTL-compatible layouts
  • Tested with Arabic, Hebrew, Persian
  • Mirror layout automatically
  • RTL-friendly navigation

To enable: Set WordPress language to RTL language (Settings → General).

Q: How do I add a language switcher?

A: Depends on your translation plugin:

WPML:

  1. Go to WPML → Languages
  2. Enable language switcher
  3. Choose position (header, footer, widget)
  4. Customize appearance

Polylang:

  1. Go to Appearance → Widgets
  2. Add Language Switcher widget
  3. Choose display format
  4. Position in header or footer

Q: Can I have different layouts per language?

A: Yes! Using WPML or Polylang:

  1. Create separate pages per language
  2. Use different page templates per language
  3. Configure unique theme options per language
  4. Different menus per language
  5. Language-specific widgets

Most translation plugins support per-language customization.

Content & Pages

Q: How do I create a homepage?

A: Two options:

Option 1: Static Homepage

  1. Create new page titled “Home”
  2. Build with page builder
  3. Go to Settings → Reading
  4. Select “A static page”
  5. Choose your Home page

Option 2: Blog Homepage

  1. Go to Settings → Reading
  2. Select “Latest posts”
  3. Blog displays on homepage

Q: How do I add a blog to my website?

A: Blog is built-in:

  1. Go to Posts → Add New
  2. Write post content
  3. Add featured image
  4. Choose category
  5. Publish

Posts automatically appear in blog section.

Q: How do I organize content with categories?

A: Create categories for posts:

  1. Go to Posts → Categories
  2. Click Add New Category
  3. Enter name and description
  4. When creating post, select category
  5. Posts grouped by category

Q: Can I have sub-categories?

A: Yes! When creating category:

  1. Enter category name
  2. Choose Parent Category from dropdown
  3. Creates sub-category hierarchy

Q: What’s the difference between Posts and Pages?

A:

  • Posts - Blog articles, show in blog section, can have categories/tags
  • Pages - Static content, don’t appear in blog, good for About/Contact

Q: Can I schedule a post to publish later?

A: Yes!

  1. When editing post
  2. Find Publish button (right side)
  3. Click Edit next to date
  4. Set future date and time
  5. Click Schedule

Post publishes automatically at scheduled time.

Q: How do I password protect a page?

A: When editing page/post:

  1. Find Publish section
  2. Click Edit next to visibility
  3. Choose Password protected
  4. Enter password
  5. Click OK

Visitors must enter password to see content.

Performance

Q: My site is slow, what can I do?

A: Try these optimization steps:

  1. Use Caching Plugin - WP Super Cache or W3 Total Cache
  2. Optimize Images - Compress before uploading
  3. Limit Plugins - Only use necessary plugins
  4. Use CDN - Distribute content globally
  5. Clean Database - Remove unused posts/media
  6. Disable Unused Features - Turn off features you don’t need

See Performance Tips for detailed guide.

Q: How do I optimize images?

A:

  1. Resize to appropriate dimensions (not larger than 1920px wide)
  2. Compress before uploading (use TinyPNG, ImageOptim)
  3. Use appropriate format:
    • Photos: JPG
    • Graphics: PNG
    • Modern browsers: WebP
  4. Add alt text for SEO

Q: What’s the best caching plugin?

A: Popular options:

  • WP Super Cache - Simple, effective
  • W3 Total Cache - Powerful, advanced
  • WP Rocket - Premium, user-friendly
  • LiteSpeed Cache - If using LiteSpeed hosting

All work well with SubNews.

Q: Should I use a CDN?

A: Yes, if:

  • You have visitors worldwide
  • You have lots of images/media
  • Want to reduce server load

Popular CDNs: Cloudflare, MaxCDN, Bunny CDN

Security

Q: Is SubNews secure?

A: Yes! SubNews includes:

  • Input sanitization
  • Output escaping
  • Nonce verification
  • SQL injection prevention
  • XSS protection

See Security Best Practices for more.

Q: How do I keep my site secure?

A: Follow these steps:

  1. Keep WordPress Updated - Always update core, plugins, themes
  2. Use Strong Passwords - 12+ characters, mix of letters/numbers/symbols
  3. Regular Backups - Backup daily/weekly
  4. Security Plugin - Use WordPress security plugin
  5. Limit Login Attempts - Prevent brute force attacks
  6. Two-Factor Auth - Enable 2FA for admin accounts
  7. Remove Unused Plugins - Less code = fewer vulnerabilities

Q: How often should I backup?

A: Backup frequency depends on update frequency:

  • High-traffic sites - Daily backups
  • Regular blogs - Weekly backups
  • Rarely updated - Monthly backups

Test backup restoration monthly.

Q: What’s the best backup plugin?

A: Popular options:

  • UpdraftPlus - Free and premium
  • BackWPup - Free, powerful
  • Duplicator - Free, great for migrations
  • BackupBuddy - Premium, automatic

All work with SubNews.

Technical Issues

Q: I see “Parse error” or “Syntax error”

A:

  1. Check PHP version (must be 7.4+)
  2. Check memory limit (must be 256MB+)
  3. Deactivate plugins except SubNews and required ones
  4. Switch to default WordPress theme temporarily
  5. Enable debug mode (see debug guide)
  6. Check debug log: /wp-content/debug.log

Q: Shortcodes display as plain text

A:

  1. Make sure Mesh Core plugin is activated
  2. Refresh page and clear browser cache
  3. Disable caching plugins temporarily
  4. Check shortcode syntax is correct
  5. Try in Visual Composer to verify

Q: Page builder not loading

A:

  1. Check WPBakery is activated
  2. Try deactivating browser extensions
  3. Clear browser cache completely
  4. Try different browser
  5. Check memory limit
  6. Check PHP version

Q: Images not showing on website

A:

  1. Check image URL is correct
  2. Verify image file exists on server
  3. Check file permissions (644 or 755)
  4. Try uploading image again
  5. Check image size (might be too large)
  6. Verify media folder exists

Q: Contact form not sending emails

A:

  1. Check email address in form settings
  2. Check email in Contact Form 7
  3. Check spam/junk folder
  4. Test with different email address
  5. Contact hosting provider if still fails

Q: Menu items not showing

A:

  1. Check menu is created
  2. Check menu items are added
  3. Check menu is assigned to location
  4. Check parent items aren’t hidden
  5. Clear browser cache
  6. Check CSS/JavaScript isn’t blocking menu

Q: Theme customizer not working

A:

  1. Check Redux Framework is activated
  2. Refresh dashboard
  3. Try different browser
  4. Clear browser cache completely
  5. Deactivate plugins temporarily
  6. Check memory limit is sufficient
