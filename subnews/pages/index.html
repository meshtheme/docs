Page Options

Page Options let you override global theme settings on a per‑page basis. You can control layout, header behavior, sidebars, and page‑level styling.

Where to find: Edit any page → look for the Page Options box.

Table of contents

  1. Page Options - General
  2. Page Options - Header
  3. Page Options - Footer
  4. Page Options - Sidebar
  5. Page Options - Design

Page Options - General

  1. Subtitle - Optional subtitle shown with the page title.
  2. Page Layout - Overrides the global layout for this page only.
  3. Featured Image (extended) - Upload multiple media items for sliders or rich media.
    Supports images (jpg, jpeg, png, gif, webp, bmp), audio (mp3, m4a, ogg, wav, wma), and video (webm, ogg, ogv, mp4).
  4. Featured Video - Paste a supported embed URL (YouTube, Vimeo, Twitter, Instagram, SoundCloud, etc.).
    Supported services: WordPress Embeds.
  5. External Link - Adds a link above the page content (useful for linking to external resources).

Page Options - Header

  1. Topbar - Show or hide the topbar for this page.
  2. Header - Show or hide the header for this page.
  3. Transparent Header - Enable an absolute, transparent header for this page.
  4. Header Background Color - Override header background color for this page.
  5. Header Text Color - Override header text color for this page.
  6. Header Background Image - Upload an image or paste a URL.
  7. Titlebar - Show or hide the page title/breadcrumb area.
  8. Titlebar Background Color - Override the titlebar background color.
  9. Titlebar Background Image - Upload an image or paste a URL.

Page Options - Footer

  1. Footer Top - Show or hide the top footer area for this page.

Page Options - Sidebar

  1. Primary Sidebar - Overrides the primary sidebar for this page.
    To hide sidebars, choose No Sidebar in Page Layout.
  2. Secondary Sidebar - Overrides the secondary sidebar for this page.
    This appears only when the layout is set to Both Sidebar.

Page Options - Design

  1. Page Background Color - Override the page background color.
  2. Page Background Image - Upload an image or paste a URL.
  3. Page Text Color - Override the page text color.
  4. Page Content Spacing (Padding) - Adds padding in pixels to all sides.
  5. Title Background - Override the page title background color.
  6. Title Text Color - Override the page title text color.
