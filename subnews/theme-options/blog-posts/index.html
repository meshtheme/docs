Blog Posts Page

Global blog and archive options that apply to the Posts page, category/tag archives, author/date archives, and blog elements.

Posts page

Note: Assign a Posts page in Settings → Reading → Posts page.
These options affect the Posts page and all archive pages (category, tag, author, date), plus blog shortcodes.

  1. Blog Posts & Archive Page Layout - Sidebar layout for all archive pages.
  2. Posts Page Max‑Width - Max width for the posts page (e.g., 700px).
  3. Posts Layout - Layout style (Big Title, Grid, Masonry, etc.).
  4. First Entity Featured - Feature the first post (not available in Masonry).
  5. Posts Grid Columns - Columns for Grid/Masonry layouts.
  6. Posts Title Spacing (Padding) - Big Title layout only.
  7. Posts Title Background - Big Title layout only.
  8. Posts Box Default Background - Default post card background.
  9. Sticky Posts Box Background - Background color for sticky posts.
  10. Gap Between Posts Entry (Margin) - Vertical spacing between items.
  11. Posts Text Spacing (Padding) - Inner padding for post content.
  12. Posts Shadow - Box shadow settings.
  13. Posts Border Radius (px) - Corner radius for post cards.
  14. Default Featured Image - Fallback when no featured image exists.
  15. Pagination Style - Pagination type for archives.
  16. Load More Button Text - Only shown when Pagination = Load More.
  17. Blog Content Display - Excerpt or full content.
  18. Content Excerpt Length - Word count for excerpts.
  19. Content – Number of Lines - Line clamp (grid view only).
  20. Read More Text - Custom label (HTML supported).
  21. Display Meta Fields - Choose which meta fields show.
  22. Show Date or Mins Read - Grid view only.
  23. Blog Filters Widgets - Select a widget to enable the filter button.

Archive & Posts Page Top Text area

  1. Archive & Posts Page Top Text - Appears above the posts list (shortcodes supported).
  2. Posts Archive Page Top Text Background - Background color/image.
  3. Posts Archive Page Top Text Color - Text color.
  4. Posts Archive Page Top Text Width - Max width for the text container.
  5. Posts Archive Page Top Text Spacing (Padding) - Padding around the text.

Single Post

  1. Single Posts Layout - Sidebar layout for single posts.
  2. Single Post Max‑Width - Max width for single posts.
  3. Show Post Title - Toggle the title display.
  4. Show Next/Previous Post Navigation - Toggle navigation links.
  5. Posts Media Border Radius (px) - Rounded corners for media.

Author Box

Note: The author name links to the author archive until a website URL is set in the user profile.

  1. Show Author Bio - Show or hide the author bio.
  2. Author Box Layout - Layout style.
  3. Author Box Text Color - Text color.
  4. Author Box Background - Background color.
  5. Author Box Shadow - Shadow settings.
  6. Author Box Border Radius (px) - Rounded corners.

Related Posts

  1. Show Related Posts - Toggle related posts.
  2. Title - Custom section title.
  3. Related Posts Filter (Type) - By Category or By Tags.
  4. Related Posts Order By - Sorting method.
  5. Related Posts Order - Ascending or descending.
  6. Number of Posts to Display - Total items.
  7. Number of Columns - Items per row.
  8. Show Next/Previous (Ajax) - Toggle AJAX navigation.
  9. Post Item Height - Height for related post cards.
  10. Border Radius - Rounded corners.

Posts Page & Archive Pages Typography

  1. Post Title - Typography for archive titles.
  2. Post Meta - Typography for archive meta.
  3. Post Text - Typography for archive excerpts.

Single Post (Details) Page Typography

  1. Post Title - Typography for single post titles.
  2. Post Meta - Typography for single post meta.
  3. Post Text - Typography for single post content.
