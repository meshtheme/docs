Installation Steps

Step 1: Download SubNews Theme

  1. Purchase SubNews from the theme provider
  2. Download the theme ZIP file to your computer
  3. Keep the ZIP file name as-is (usually subnews.zip)

Step 2: Install the Theme

Via WordPress Dashboard:

  1. Log in to your WordPress admin panel
  2. Go to Appearance ‚Üí Themes
  3. Click Add New button
  4. Click Upload Theme button
  5. Choose the SubNews ZIP file from your computer
  6. Click Install Now
  7. Wait for installation to complete
  8. Click Activate

Or via FTP:

  1. Extract the SubNews ZIP file
  2. Upload the subnews folder to /wp-content/themes/ via FTP
  3. Go to Appearance ‚Üí Themes in WordPress
  4. Find SubNews and click Activate

Step 3: Verify Installation

After activation:

  1. Visit your website homepage
  2. You should see the SubNews theme active
  3. Check that the theme displays correctly

‚úÖ Success! Your theme is now installed.

Troubleshooting Installation

Problem: Theme Not Showing

Solution:

  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Go to Appearance ‚Üí Themes
  3. Verify SubNews is activated
  4. Check WordPress version (must be 5.8+)

Problem: Plugins Won‚Äôt Install

Solution:

  1. Check your hosting memory limit (must be 256MB+)
  2. Try installing one plugin at a time
  3. Wait 30 seconds between plugin installations
  4. Contact hosting provider if problem persists

Problem: Blank Page After Installation

Solution:

  1. Enable WordPress Debug (see troubleshooting guide)
  2. Check error logs in /wp-content/debug.log
  3. Disable all plugins except SubNews and required ones
  4. Switch to default theme and back to SubNews

Problem: Missing Theme Options Menu

Solution:

  1. Make sure Redux Framework is activated
  2. Go to Plugins ‚Üí make sure it shows as Active
  3. Deactivate and reactivate Redux Framework
  4. Refresh WordPress dashboard
