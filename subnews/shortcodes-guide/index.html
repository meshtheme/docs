What Are Shortcodes?

Shortcodes are simple codes that display complex content. Instead of coding, you use simple shortcuts.

Shortcode Syntax

Shortcodes look like this:

[mtc_news_block columns="3" posts="9"]

Two Ways to Use Shortcodes

1. Visual Composer (Page Builder)

  • Choose element
  • Configure settings visually
  • Easy for beginners

2. Direct Code (Advanced)

  • Copy shortcode
  • Paste into page content
  • Edit parameters
  • Powerful and flexible

How to Add Shortcodes

Method 1: Using Page Builder (Easiest)

  1. Edit or create a page
  2. Click Visual Composer tab
  3. Click Add button
  4. Search for element name
  5. Click to add element
  6. Configure element settings
  7. Save changes

Method 2: Add Element Directly

  1. Edit page content area
  2. Click Add Element button
  3. Choose element category
  4. Select specific element
  5. Fill in element parameters
  6. Save page

Method 3: Direct Shortcode (Advanced)

  1. Click Text tab in editor
  2. Paste shortcode code
  3. Edit parameters as needed
  4. Save page

Content Elements

News Block

Display posts in beautiful grid layouts.

Visual Composer Settings:

  • Columns - Number of columns (1-4)
  • Posts per page - How many posts to show
  • Layout - Choose from 9 different styles
  • Category - Filter by category
  • Order by - Latest, Popular, Random
  • Title length - Character count for titles
  • Content length - Character count for excerpts

Usage Example:

[mtc_news_block columns="3" posts="9" layout="style1" category="news"]

Post Carousel

Display posts in a scrolling carousel.

Settings:

  • Posts per page - Number of posts
  • Items visible - Posts visible at once
  • Autoplay - Auto-scroll on/off
  • Autoplay speed - Scroll speed (milliseconds)
  • Category - Filter by category
  • Layout - Visual style

Best for:

  • Featured posts
  • Latest content
  • Category showcases

Featured Block

Highlight important content.

Settings:

  • Title - Block heading
  • Description - Block text
  • Background - Custom background color/image
  • Layout - Visual style (9 options)

Best for:

  • Promotions
  • Important announcements
  • Feature highlights

Tips for Using Shortcodes

Best Practices

  1. Use Visual Composer for easier configuration
  2. Preview before publishing to check appearance
  3. Optimize images before adding to shortcodes
  4. Test on mobile to ensure responsive design
  5. Use appropriate element for your content type

Common Issues

Shortcode displays as text instead of element:

  • Make sure plugin is activated
  • Refresh page
  • Clear browser cache

Element looks broken:

  • Check image URLs
  • Verify all required fields filled
  • Check for special characters in text

Carousel not scrolling:

  • Enable JavaScript
  • Check autoplay settings
  • Verify responsive settings

Responsive Behavior

All shortcodes are responsive and adapt to device size:

  • Desktop - Full-width, multiple columns
  • Tablet - Reduced columns, readable text
  • Mobile - Single column, touch-friendly
  • Test - Use browser DevTools to test

Performance Tips

  1. Limit elements per page - Use 5-10 shortcodes max
  2. Optimize images - Compress before use
  3. Use carousels sparingly - Can slow down page
  4. Lazy load images - Use image optimization
  5. Cache pages - Use caching plugin
