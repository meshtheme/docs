Overview

SubNews is a modern, feature-rich WordPress theme designed for news, magazine, and content-focused websites. It includes extensive customization options through Redux Framework, deep Visual Composer integration, and a companion plugin (mesh-core) for advanced functionality.

Key Features

  • Responsive Design - Bootstrap 5.3.2 framework
  • Multiple Layouts - 6 header layouts, 24 blog block variations
  • Visual Page Builder - WPBakery Page Builder integration with 40+ custom elements
  • Theme Options - Redux Framework with 22 configuration sections
  • Metabox System - CMB2 for page/post customization
  • AJAX Features - Load more, block pagination, search
  • WooCommerce Ready - Full e-commerce support
  • Portfolio Support - Custom post type via mesh-core
  • Security First - Nonces, capabilities, sanitization throughout

Statistics

  • 151 PHP files - Organized in includes/ subdirectory
  • 10,094+ lines - Core functionality code
  • 25+ classes - Object-oriented architecture
  • 32 template parts - Reusable template components
  • 19 action hooks - WordPress hook integration points
  • 22+ filter hooks - Data modification and customization
  • 5 menu locations - Navigation registration
  • 7+ widget areas - Dynamic sidebar support
