Overview

Mesh-Core is a comprehensive WordPress plugin that extends MeshTheme WordPress themes with advanced functionality. It provides a library of 40+ custom Visual Composer elements (shortcodes), custom post types, widgets, and social media integration.

Key Features

  • 40+ Visual Composer Elements - Pre-built page building components
  • 34+ Shortcodes - Versatile content elements
  • Custom Post Type - Portfolio management
  • 3 WordPress Widgets - Tab widget, posts widget, author widget
  • View Counter System - Track post popularity
  • Social Media Integration - Share buttons and social links
  • WooCommerce Support - Product carousels and shortcodes
  • Performance Hooks - Track template loading performance
  • WPML Compatible - Multilingual support
  • Security First - Input sanitization, nonce verification

Statistics

  • 40+ shortcode elements - Full page building library
  • 34+ shortcodes - Direct content insertion
  • 2,553 instances - Sanitization functions
  • 77 layout preview images - Admin interface previews
  • 90+ section separators - Design variations
  • 1 CPT, 1 taxonomy - Content organization
  • 3 WordPress widgets - Sidebar functionality

Shortcode Architecture Pattern

Each element class follows this pattern:

class Mesh_Element_Name {
    public function __construct() {
        // Hook into VC
        if ( function_exists( 'vc_add_shortcode_param' ) ) {
            add_action( 'vc_before_init', array( $this, 'mapping' ) );
        }

        // Register shortcode
        add_shortcode( 'mtc_element', array( $this, 'render' ) );
    }

    public function mapping() {
        // VC element mapping
        vc_map( array(
            'name'        => 'Element Name',
            'base'        => 'mtc_element',
            'icon'        => 'icon-url',
            'category'    => 'Category',
            'params'      => array( /* parameters */ ),
        ) );
    }

    public function render( $atts, $content = null ) {
        // Shortcode output
        $atts = shortcode_atts( array(
            'param1' => 'default',
            'param2' => 'default',
        ), $atts );

        // Sanitization & escaping
        $param1 = sanitize_text_field( $atts['param1'] );

        // Output
        $html = '<div class="element">' . wp_kses_post( $content ) . '</div>';
        return $html;
    }
}
new Mesh_Element_Name();

Performance Testing

add_action( 'start_operation', function( $handle ) {
    global $perf;
    $perf[ $handle ] = microtime( true );
} );

add_action( 'end_operation', function( $handle ) {
    global $perf;
    if ( isset( $perf[ $handle ] ) ) {
        $time = microtime( true ) - $perf[ $handle ];
        error_log( $handle . ' took ' . $time . ' seconds' );
    }
} );

Troubleshooting

Common Issues

Shortcodes not appearing:

  • Check if Visual Composer is active
  • Verify element class is loaded
  • Check for PHP errors in debug log

AJAX not working:

  • Check nonce is being sent
  • Verify admin-ajax.php is accessible
  • Check browser console for errors

Widget not displaying:

  • Verify widget is registered
  • Check sidebar is active
  • Verify theme supports widgets
