Create a Blog Post

Step 1: Go to Posts

  1. Log in to WordPress dashboard
  2. Click Posts in left menu
  3. Click Add New

Step 2: Write Post Content

  1. Enter Post Title at top
  2. Click in content area
  3. Type or paste post content

Step 3: Add Featured Image

  1. On right side, find Featured Image
  2. Click Set featured image
  3. Upload new image or choose from library
  4. Click Set featured image button

Step 4: Set Categories

  1. On right side, find Categories
  2. Check the categories for this post
  3. Or click Add New Category to create new one

Step 5: Add Tags (Optional)

  1. On right side, find Tags
  2. Type tag names, separated by commas
  3. Tags help organize posts

Step 6: Set Post Options (Optional)

  1. Scroll down to find metabox Post Options
  2. Set:
    • Subtitle - Shows under title
    • Post Layout - Sidebar options
    • Custom background or styling
  3. These override theme defaults for this post

Step 7: Preview and Publish

  1. Click Preview to see how it looks
  2. If looks good, click Publish
  3. Your post is now live!

Editing Existing Post

  1. Go to Posts
  2. Hover over post title
  3. Click Edit
  4. Make changes
  5. Click Update

Create a Page

Step 1: Create New Page

  1. Go to Pages in left menu
  2. Click Add New

Step 2: Add Page Title

  1. Enter page title
  2. This appears at top of page

Step 3: Build Page Content

Option A: Use Page Builder (Recommended)

  1. Click Visual Composer tab
  2. Click Add button
  3. Choose elements
  4. Configure each element
  5. Build your layout

Option B: Use Text Editor

  1. Click Text tab
  2. Type content or paste shortcodes
  3. Format using toolbar

Step 4: Set Page Template (Optional)

  1. On right side, find Page Attributes
  2. Choose template:
    • Default
    • Full Width
    • Blank
  3. Select one appropriate for your page

Step 5: Customize Page (Optional)

  1. Scroll down to Page Options metabox
  2. Customize:
    • Custom header layout
    • Custom background
    • Custom styling
  3. Save changes

Step 6: Publish

  1. Click Preview to check
  2. Click Publish when ready
  3. Your page is live!

Important: Set Page as Homepage

  1. Go to Settings ‚Üí Reading
  2. Select A static page
  3. Choose your page from dropdown
  4. Save changes

Add to Navigation Menu

Step 1: Create Menu Item

  1. Go to Appearance ‚Üí Menus
  2. Click Create New Menu
  3. Enter menu name
  4. Click Create Menu

Step 2: Add Items to Menu

  1. On left side, find categories:
    • Pages - Add pages to menu
    • Posts - Add posts to menu
    • Categories - Add categories to menu
    • Custom Links - Add external links
  2. Check items you want to add
  3. Click Add to Menu

Step 3: Organize Menu Items

  1. Drag items up/down to order
  2. Drag items right to create sub-menu
  3. Items indented = sub-menu items

Step 4: Assign to Location

  1. Scroll to Display location
  2. Check menu locations:
    • Primary Menu - Main header menu
    • Secondary Menu - Alternative menu
    • Topbar Menu - Top bar menu
    • Side Menu - Mobile menu
  3. Click Save Menu

Step 5: Verify

  1. Visit your website
  2. Check that menu appears
  3. Verify links work correctly
  4. Check sub-menus open

Editing Existing Menu

  1. Go to Appearance ‚Üí Menus
  2. Select menu to edit
  3. Make changes:
    • Add/remove items
    • Reorder items
    • Edit item labels
  4. Click Save Menu

Create Portfolio Item

Step 1: Go to Portfolio

  1. Go to Portfolio in left menu
  2. Click Add New

Step 2: Add Project Details

  1. Enter Project Title
  2. Enter Project Description
  3. This is like a blog post description

Step 3: Add Featured Image

  1. On right, click Set featured image
  2. Upload or choose image
  3. This displays in portfolio listing

Step 4: Add Portfolio Information

  1. Scroll down to Portfolio Options metabox
  2. Fill in:
    • Project Date - When project was completed
    • Client Name - Who you did it for
    • Skills/Technologies - What you used
    • Project URL - Link to live project (if online)
  3. Add additional images if needed

Step 5: Assign Category

  1. On right, find Portfolio Categories
  2. Check category for this project
  3. Or create new category

Step 6: Publish

  1. Click Preview to check
  2. Click Publish
  3. Portfolio item now appears in portfolio section

Set Up Contact Form

Step 1: Install Contact Form 7

  1. Go to Plugins ‚Üí Add New
  2. Search ‚ÄúContact Form 7‚Äù
  3. Click Install Now
  4. Click Activate

Step 2: Create Form

  1. Go to Contact in left menu
  2. Click Add New
  3. Enter form title
  4. Use form builder:
    • Click Text to add text field
    • Click Email to add email field
    • Click Textarea to add message field
    • Click Submit to add submit button
  5. Save form

Step 3: Add to Page

  1. Copy shortcode from form
  2. Edit page/post where you want form
  3. Paste shortcode into content
  4. Publish page

Step 4: Configure Email

  1. Go to form in Contact menu
  2. Click Mail tab
  3. Set:
    • To - Where emails go
    • From - From address
    • Subject - Email subject
  4. Save

Step 5: Test

  1. Go to page with form
  2. Fill out form
  3. Click Submit
  4. Check that email arrives

Add WooCommerce Product

Step 1: Go to Products

  1. Go to Products in left menu
  2. Click Add New

Step 2: Add Product Details

  1. Enter Product Title
  2. Enter Product Description
  3. Explain what product does

Step 3: Upload Product Image

  1. On right, click Set featured image
  2. Upload product photo
  3. This shows in product listing

Step 4: Set Price

  1. Scroll down to Product Data
  2. Choose product type:
    • Simple - Single product
    • Variable - Different options (sizes, colors)
  3. Enter Price
  4. Set Sale Price (optional)

Step 5: Set Inventory

  1. In Product Data, click Inventory tab
  2. Set:
    • Stock quantity - How many in stock
    • Track stock - Yes/No
    • Low stock threshold - When to reorder

Step 6: Add Categories

  1. On right, find Product Categories
  2. Check relevant categories
  3. Or create new category

Step 7: Publish

  1. Click Publish
  2. Product now appears in shop

Change Site Colors

Step 1: Go to Theme Options

  1. Go to Appearance ‚Üí SubNews Theme Options
  2. Click Colors section

Step 2: Change Primary Color

  1. Click on Primary Color box
  2. Choose new color from picker
  3. Or enter hex code (like #FF5733)

Step 3: Change Secondary Color

  1. Click on Secondary Color box
  2. Choose color
  3. This color used for accents

Step 4: Change Text Color

  1. Find Text Color
  2. Choose new color
  3. Used for body text

Step 5: Save Changes

  1. Click Save Changes button
  2. Wait for confirmation
  3. Visit website to see changes

Undo Changes

  1. Go back to Colors
  2. Click default colors
  3. Click Save Changes

Change Header Layout

Step 1: Go to Theme Options

  1. Go to Appearance ‚Üí SubNews Theme Options
  2. Click Header section

Step 2: Choose Layout

  1. Find Header Layout option
  2. Choose from 6 layouts:
    • Layout 1: Standard
    • Layout 2: Minimal Centered
    • Layout 3: Full Width Search
    • Layout 4: Magazine Style
    • Layout 5: Search Focused
    • Layout 6: Mega Menu
  3. Click layout you want

Step 3: Configure Header

  1. Set Header Height (optional)
  2. Enable Sticky Header if desired
  3. Enable Transparent Header if desired
  4. Upload Custom Logo (optional)

Step 4: Save

  1. Click Save Changes
  2. Visit website
  3. See new header layout

Add Image to Media

Step 1: Go to Media

  1. Go to Media in left menu
  2. Click Add New

Step 2: Upload Image

  1. Click Select Files button
  2. Choose image from computer
  3. Or drag image into area
  4. Wait for upload to complete

Step 3: Edit Image (Optional)

  1. Click image to edit
  2. Change:
    • Title
    • Caption
    • Description
    • Alt text

Step 4: Save

  1. Click Update
  2. Image available in media library

Using Image in Post/Page

  1. Click Image Icon in editor
  2. Choose image from library
  3. Configure:
    • Size
    • Link destination
    • Alignment
  4. Click Insert
