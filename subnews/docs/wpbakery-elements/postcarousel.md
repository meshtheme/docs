### Post Carousel - General

![Image](/subnews/assets/images/vc/post-carousel-general.jpg)
{: .clx-shadow }

1. Title - Optional heading for the carousel.
2. Total Number of Posts to Show - Number of posts to include.
3. Number of Columns - Number of items visible at once.
4. Enable Carousel Arrow - Show or hide navigation arrows.
5. Enable Carousel Indicators - Show or hide dot indicators.
6. Carousel Draggable - Enable drag/swipe navigation.
7. Carousel Loop - Loop through posts continuously.
8. Carousel Image Lazy Load - Load images as they appear.
9. Carousel Autoplay - Enable automatic sliding.
10. Carousel Interval/Autoplay Timeout - Delay in milliseconds (default: 7000).
11. Carousel Item Spacing (Margin) - Spacing between items (e.g. `10`).
12. Carousel Stage Padding - Padding on the left/right edges.
13. Direction from Right to Left - Enable RTL direction.
14. Item Height - Fixed item height in pixels (leave blank for auto).
15. Link - Optional URL to link the title or carousel.
{: .clx-label }
### Post Carousel - Data Options

![Image](/subnews/assets/images/vc/post-carousel-data-options.jpg)
{: .clx-shadow }

1. Offset - Skip posts before showing results (e.g. `5` to start from the 6th post).
2. Include Category - Select one or more categories. To select multiple, hold Control (Windows) or Command (Mac).
3. Tags - Use comma-separated tags for """any of""" or plus-separated tags for """all of""".
4. Include/Exclude Author by ID - Include IDs (e.g. `2,6,17,38`) or exclude by prefixing with `-` (e.g. `-12`).
5. Order By - Sort by the selected parameter.
6. Order - Ascending or descending (default: descending).
{: .clx-label }
