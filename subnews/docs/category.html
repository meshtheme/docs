Category Page Options

Table of contents

  1. Category Image - Category Thumbnail image for this category (gif, jpg, or png images).
  2. Featured/Hero Layout - This optional will override the global layout only for this category page.(Chillax Theme Options -> Category -> Hero Layout).
  3. Posts Layout/Style - This optional will override the global layout only for this page.(Chillax Theme Options -> Blog Posts -> Posts Layout).
  4. Title Bar - This page title bar/breadcrumb visibility option.
