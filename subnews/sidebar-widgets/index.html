Sidebar Widgets (Custom Widget Areas)

SubNews includes a Sidebar Widgets manager (via the Mesh Core plugin) so you can create custom widget areas and assign them to specific pages or layouts.

Create a custom sidebar

  1. Go to SubNews → Sidebar Widgets (Widgets Area Manager).
  2. Click + Create New Sidebar.
  3. Enter a name (for example: “Shop Sidebar” or “About Page Sidebar”).
  4. Click Create.

Your new sidebar appears in the list with a generated Slug.

Add widgets to your sidebar

  1. Go to Appearance → Widgets.
  2. Find your new sidebar by name.
  3. Add the widgets you want.

Assign the sidebar to a page

You can assign custom sidebars per page:

  • Edit a page → Page Options → Sidebar
  • Choose your custom sidebar for Primary Sidebar (and Secondary Sidebar if using “Both Sidebar” layout)

See: Page Options

Set a global sidebar

To change sidebars globally, use:

Theme Options → Sidebar See: Theme Options → Sidebar

Notes and tips

  • Built‑in sidebars are shown in the list and cannot be deleted.
  • Custom sidebars can be deleted from the manager (this does not delete widgets from your site, but the area will be removed).
  • Sidebar names must be unique and cannot be empty.

Troubleshooting

My new sidebar doesn’t show in Widgets
Make sure the Mesh Core plugin is active, then reload Appearance → Widgets.

Sidebar not appearing on the page
Check that the page layout isn’t set to No Sidebar and that the correct sidebar is selected in Page Options.
